virtual HRESULT __stdcall CubeMelon::IIOComponent::Close ( IComponent *  listener = nullptr)
pure virtual

所有しているオブジェクトを閉じます。

無事閉じた場合、コンポーネントの状態からは STATE_OPEN フラグが取り除かれます。
結果は非同期でも受け取ることができます。

引数
[in]listener結果を非同期で受け取る場合の通知先オブジェクト

結果を非同期で受け取る場合、Notifyメッセージのパラメータは以下のようになります。
  • [in] sender このコンポーネント
  • [in] msg "IIOComponent.Close()"
  • [inout] data HRESULT型変数へのポインタ
  • [in] cb_data sizeof(HRESULT)
戻り値
無事閉じられたか
戻り値
S_OKオブジェクトは無事閉じられた
S_OK非同期処理を開始した
S_FALSE既に閉じている
E_FAILオブジェクトを閉じられなかった
E_FAIL非同期処理を開始できなかった

CubeMelon::Wave, と CubeMelon::Wasapiで実装されています。
